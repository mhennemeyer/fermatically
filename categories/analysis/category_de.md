# Analysis

Die Analysis untersucht Funktionen, Grenzwerte, Stetigkeit und Konvergenz. Von den reellen Zahlen über komplexe Funktionentheorie bis hin zu Modulformen – analytische Methoden sind unverzichtbar für das Verständnis tiefer zahlentheoretischer Zusammenhänge.
